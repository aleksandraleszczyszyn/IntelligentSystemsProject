◆ sumWeight

float Cluster::sumWeight
protected

The sum of the global weights is maintained along with the sum of the (weight * feature) calculation

This is done so that updating the centroids is less computationally intensive

The centroid is calculated by dividing the sumFeatures by the sumWeight (sumFeatures / sumWeight)

When a new feature is added to the cluster its weight s added to the sum of weights (sumWeight) The product (weight * feature) is also added to the weighted sum of the features (sumFeatures)

To remove the feature from the cluster the process is similar, but instead of adding to the running summations, we subtract
